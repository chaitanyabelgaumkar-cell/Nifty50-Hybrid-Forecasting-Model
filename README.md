# Modeling and Forecasting Nifty 50 using Hybrid ARIMA-GARCH Model

## Project Overview
This repository contains my MBA dissertation project focused on the econometric modeling of the **Nifty 50 Index**. The study addresses the limitations of traditional linear models by implementing a **Hybrid ARIMA-GARCH** approach to capture both mean trends and conditional heteroskedasticity (volatility clustering) in Indian equity markets.

## Key Objectives
* Analyze the historical performance of the Nifty 50 Index (2015–2025).
* Test for stationarity and remove non-linear trends in financial time series data.
* Develop a robust forecasting model that accounts for market volatility and "shocks."

## Methodology & Tools
* **Econometric Modeling:** ARIMA (for mean equation) + GARCH (for variance equation).
* **Software:** **EViews** (Estimation, Residual Diagnostics, and Forecasting).
* **Statistical Tests:** Augmented Dickey-Fuller (ADF) Test, ARCH-LM Test for volatility effects.

## Repository Structure
* `/Chaitanya Nifty50.pdf`: The full dissertation report.
* `/Data`: Contains the raw Nifty 50 dataset.
* `/EViews-Workfile`: The `.wf1` file containing the model objects and equations.
